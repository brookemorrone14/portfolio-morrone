Skills I've Learned

Illustrator

Graphics/Visual Software

I LOVE Illustrator. It is 100% my go-to program to design in, and arguably the most fun. All the adobe tools are pretty straight-forward and easy-to-use.

InDesign

Type/Layout Software

I wasn't a huge fan of InDesign when I first joined the graphic design program, but it's growing on me. It's very technical which I like, but also VERY complicated.

Photoshop

Photo-editing Software

Photoshop is SO cool. It has so many features that I had no idea were possible, like smart-selecting and making smart layers. I use it in almost every project.

Figma

Visual/Coding Software

I think Figma is a universal software for designers, and it's going to become the main go-to program. It has visual design tools AND web tools, which I love.

Procreate

Drawing Software

Procreate is the only graphic design software that is self-taught for me. I have an iPad, and naturally got Procreate with it, it makes drawing so much easier.
